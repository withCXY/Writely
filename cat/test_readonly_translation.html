🌐 Read-only Text Translation Test

✨ Feature Overview:

This page tests the read-only text translation feature. When you select text in read-only areas, the AI Assistant should:

  • 🔍 Detect read-only text and show appropriate menu options
  • 🌐 Show translation submenu with language options
  • 📋 Display translation result in a popup window (not replace original text)
  • 📄 Provide copy button to copy translation to clipboard

📖 Read-only Text Behavior

  • Shows "Translate ▸" and "Close" only
  • Translation result appears in popup
  • Original text remains unchanged
  • Copy button provided

✏️ Editable Text Behavior

  • Shows full menu (Translate, Rewrite, Tone)
  • Translation directly replaces text
  • Original text is modified
  • No popup window
