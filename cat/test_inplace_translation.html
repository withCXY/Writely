🔍 Technical Implementation

The new replaceSelectedTextInPlace() function uses multiple strategies:

  1. Range-based replacement: Uses saved selection range with execCommand
  2. Input/Textarea handling: Direct value manipulation for form elements
  3. ContentEditable support: innerHTML replacement for editable divs
  4. DOM text node replacement: Direct textContent manipulation for read-only text
  5. Range deleteContents/insertNode: Advanced DOM manipulation
  6. Parent innerHTML replacement: Fallback for complex structures
  7. Popup fallback: Shows translation popup if all methods fail
